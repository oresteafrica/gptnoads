Tout d'abord, obtenez une clé API pour utiliser ChatGPT dans l'application GPTNOADS en suivant ces étapes :

Étape 1 : Créer un compte OpenAI
Si vous ne l'avez pas déjà fait, visitez le site Web d'OpenAI (https://openai.com) et créez un compte. Fournissez les informations requises et complétez le processus d'inscription.

Étape 2 : accédez au tableau de bord du développeur OpenAI
Une fois que vous avez réussi à vous inscrire et à vous connecter à votre compte OpenAI, accédez au tableau de bord du développeur OpenAI. C'est ici que vous pouvez gérer vos clés API et accéder aux informations pertinentes.

Étape 3 : Créer une nouvelle clé d'API
Sur le tableau de bord du développeur, localisez la section "Clés API" et cliquez dessus. Ensuite, cliquez sur le bouton "Nouvelle clé" pour générer une nouvelle clé API spécifiquement pour votre application Android.

Étape 4 : Fournissez les détails nécessaires
Indiquez le nom suivant pour votre clé API : "GPTNOADS" pour vous aider à identifier son objectif. Si vous le souhaitez, vous pouvez spécifier les détails d'utilisation pertinents pour votre application GPTNOADS, tels que le cas d'utilisation prévu et toute autre information pertinente.

Étape 5 : Lisez et acceptez les conditions d'utilisation d'OpenAI
Assurez-vous de lire attentivement les conditions d'utilisation et la politique d'utilisation acceptable d'OpenAI. Une fois que vous avez compris et accepté les conditions, procédez en reconnaissant votre conformité.

Étape 6 : Récupérez votre clé API
Après avoir accepté les termes, votre clé API sera générée. Assurez-vous de copier et de stocker en toute sécurité cette clé, car elle donne accès à l'API ChatGPT.

Étape 7 : Intégrez la clé API dans votre application GPTNOADS
Dans le menu GPTNOADS choisissez "Clé API" puis entrez manuellement la clé.

Étape 8 : Testez et affinez votre intégration
Avec la clé API intégrée à l'application GPTNOADS, commencez à tester avec une question simple comme par exemple : « La capitale de la France » et observez les réponses. En cas de message d'erreur, vérifiez si vous avez saisi la bonne clé. Il s'agit d'un code alphanumérique long, il peut donc arriver que vous ne le saisissiez pas correctement, vérifiez la différence entre zéro (0) et la lettre O. Une fois la clé saisie, vous verrez la réponse sur votre écran.

paramètres
Le premier choix fait référence au modèle, il existe deux modèles :
gpt-4 et gpt-3.5-turbo
gpt-4 est le modèle le plus puissant mais aussi plus bas et plus cher que gpt-3.5-turbo
gpt-3.5-turbo est rapide, moins cher mais moins précis que gpt-4
Les modèles évoluent rapidement donc vaut mieux se référer aux pages officielles
Explication complète des modèles en anglais sur : https://platform.openai.com/docs/models/overview
Les explications sur les coûts en anglais se trouvent dans le même document au paragraphe "Tokens".
D'autres paramètres sont disponibles pour les professionnels souhaitant les tester. A utiliser avec prudence car il est nécessaire de savoir comment ils fonctionnent pour éviter les perturbations. Ne les utilisez pas si vous ne savez pas comment procéder.
