Prije svega nabavite API ključ za korištenje ChatGPT-a u aplikaciji GPTNOADS slijedeći ove korake:

1. korak: prijavite se za OpenAI račun
Ako već niste, posjetite web mjesto OpenAI (https://openai.com) i kreirajte račun. Unesite potrebne podatke i dovršite postupak registracije.

Korak 2: Idite na OpenAI Developer Dashboard
Nakon što ste se uspješno prijavili i prijavili na svoj OpenAI račun, idite na OpenAI Developer Dashboard. Ovdje možete upravljati svojim API ključevima i pristupiti relevantnim informacijama.

3. korak: Stvorite novi API ključ
Na nadzornoj ploči razvojnog programera pronađite odjeljak "API ključevi" i kliknite na njega. Zatim kliknite gumb "Novi ključ" za generiranje novog API ključa posebno za vašu Android aplikaciju.

4. korak: Navedite potrebne pojedinosti
Navedite sljedeći naziv za svoj API ključ: "GPTNOADS" kako biste lakše prepoznali njegovu svrhu. Ako želite, možete navesti relevantne pojedinosti o korištenju vaše aplikacije GPTNOADS, kao što je predviđeni slučaj upotrebe i sve druge relevantne informacije.

Korak 5: Pregledajte i prihvatite OpenAI uvjete
Obavezno temeljito pregledajte OpenAI uvjete pružanja usluge i prihvatljivu politiku korištenja. Nakon što ste razumjeli i prihvatili uvjete, nastavite s potvrdom svoje usklađenosti.

Korak 6: Dohvatite svoj API ključ
Nakon prihvaćanja uvjeta, vaš API ključ će biti generiran. Obavezno kopirajte i sigurno pohranite ovaj ključ jer on omogućuje pristup ChatGPT API-ju.

Korak 7: Integrirajte API ključ u svoju aplikaciju GPTNOADS
U izborniku GPTNOADS odaberite “API ključ” i zatim ručno unesite ključ.

Korak 8: Testirajte i poboljšajte svoju integraciju
S API ključem integriranim u aplikaciju GPTNOADS, započnite testiranje s jednostavnim pitanjem kao što je na primjer: "Glavni grad Francuske" i promatrajte odgovor. U slučaju poruke o pogrešci provjerite jeste li unijeli točan ključ. To je dugačak alfanumerički kod pa se može dogoditi da ga ne unesete ispravno, provjerite razliku između nule (0) i slova O. Nakon što unesete ključ, vidjet ćete odgovor na zaslonu.

parametri
Prvi izbor odnosi se na model, postoje dva modela:
gpt-4 i gpt-3.5-turbo
gpt-4 je najjači model, ali također niži i skuplji u usporedbi s gpt-3.5-turbo
gpt-3.5-turbo je brz, jeftiniji, ali je manje precizan od gpt-4
Modeli se brzo razvijaju stoga je bolje pogledati službene stranice
Potpuno objašnjenje modela na engleskom jeziku na: https://platform.openai.com/docs/models/overview
Objašnjenja o troškovima na engleskom jeziku nalaze se u istom dokumentu u odjeljku "Tokeni".
Ostali parametri dostupni su profesionalcima koji ih žele testirati. Koristite ih s oprezom jer je potrebno znati kako rade kako biste izbjegli smetnje. Nemojte ih koristiti ako niste sigurni kako to učiniti.
Potpuno objašnjenje parametara na engleskom jeziku na:
https://platform.openai.com/docs/api-reference/completions/create
