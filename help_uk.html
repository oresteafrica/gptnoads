Перш за все, отримайте ключ API для використання ChatGPT у програмі GPTNOADS, виконавши такі дії:

Крок 1. Зареєструйте обліковий запис OpenAI
Якщо ви ще цього не зробили, відвідайте веб-сайт OpenAI (https://openai.com) і створіть обліковий запис. Надайте необхідну інформацію та завершіть процес реєстрації

Крок 2. Перейдіть до інформаційної панелі розробника OpenAI
Після успішної реєстрації та входу в обліковий запис OpenAI перейдіть до інформаційної панелі розробника OpenAI. Тут ви можете керувати своїми ключами API та отримувати доступ до відповідної інформації.

Крок 3. Створіть новий ключ API
На інформаційній панелі розробника знайдіть розділ «Ключі API» і натисніть на нього. Потім натисніть кнопку «Новий ключ», щоб створити новий ключ API спеціально для вашої програми Android.

Крок 4. Надайте необхідні відомості
Укажіть таку назву для свого ключа API: "GPTNOADS", щоб допомогти вам визначити його призначення. За бажанням ви можете вказати відповідні деталі використання для своєї програми GPTNOADS, як-от запланований варіант використання та будь-яку іншу відповідну інформацію.

Крок 5. Перегляньте та погодьтеся з умовами OpenAI
Обов’язково уважно ознайомтеся з умовами обслуговування та політикою прийнятного використання OpenAI. Після того, як ви зрозуміли та погодилися з умовами, продовжуйте, підтверджуючи свою відповідність.

Крок 6. Отримайте ключ API
Після прийняття умов ваш ключ API буде згенеровано. Обов’язково скопіюйте та безпечно зберігайте цей ключ, оскільки він надає доступ до ChatGPT API.

Крок 7. Інтегруйте ключ API у свою програму GPTNOADS
У меню GPTNOADS виберіть «ключ API», а потім введіть ключ вручну.

Крок 8. Перевірте та вдосконаліть інтеграцію
За допомогою ключа API, інтегрованого в додаток GPTNOADS, почніть тестування з простого запитання, наприклад: «Столиця Франції», і спостерігайте за відповідями. У разі повідомлення про помилку перевірте, чи ви ввели правильний ключ. Це довгий буквено-цифровий код, тому може статися, що його неправильно ввести, перевірте різницю між нулем (0) і літерою O. Після введення ключа ви побачите відповідь на екрані.

Меню

історія
Відображає в області відповідей усі запитання користувача та відповіді, які були заповнені раніше.

скинути
Очистіть області питань і відповідей.

параметри
Перший вибір стосується моделі, є дві моделі:
gpt-4 і gpt-3.5-turbo
gpt-4 є найпотужнішою моделлю, але також нижчою та дорожчою порівняно з gpt-3.5-turbo
gpt-3.5-turbo швидкий, менш дорогий, але менш точний, ніж gpt-4
Моделі швидко розвиваються, тому краще звертатися до офіційних сторінок
Повне пояснення моделей англійською мовою: https://platform.openai.com/docs/models/overview
Пояснення щодо вартості англійською мовою знаходяться в цьому ж документі в параграфі «Токени».
Інші параметри доступні для професіоналів, які бажають їх перевірити. Використовуйте з обережністю, оскільки необхідно знати, як вони працюють, щоб уникнути збоїв. Не використовуйте їх, якщо ви не впевнені, як це зробити.
Повне пояснення параметрів англійською мовою за адресою:
https://platform.openai.com/docs/api-reference/completions/create

мова
Доступно 18 мов. Більше на вимогу.

Ключ API
Тут ви можете вставити свій персональний ключ API. Зверніться до пояснень угорі цього розділу довідки, щоб дізнатися, як отримати ключ API.
Сторінка, на якій можна зареєструватися:
https://openai.com/
Після входу отримайте ключ API за адресою: https://platform.openai.com/
Натисніть ім’я свого облікового запису, яке зазвичай розташоване у верхньому правому куті екрана, виберіть «Переглянути ключі API» та створіть свій ключ API.
