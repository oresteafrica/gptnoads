Πρώτα απ 'όλα αποκτήστε ένα κλειδί API για τη χρήση του ChatGPT στην εφαρμογή GPTNOADS ακολουθώντας αυτά τα βήματα:

Βήμα 1: Εγγραφείτε για λογαριασμό OpenAI
Εάν δεν το έχετε κάνει ήδη, επισκεφτείτε τον ιστότοπο του OpenAI (https://openai.com) και δημιουργήστε έναν λογαριασμό. Δώστε τις απαιτούμενες πληροφορίες και ολοκληρώστε τη διαδικασία εγγραφής.

Βήμα 2: Μεταβείτε στον Πίνακα ελέγχου προγραμματιστών OpenAI
Αφού εγγραφείτε επιτυχώς και συνδεθείτε στον λογαριασμό σας στο OpenAI, μεταβείτε στον Πίνακα ελέγχου προγραμματιστών OpenAI. Εδώ μπορείτε να διαχειριστείτε τα κλειδιά API σας και να αποκτήσετε πρόσβαση σε σχετικές πληροφορίες.

Βήμα 3: Δημιουργήστε ένα νέο κλειδί API
Στον Πίνακα ελέγχου προγραμματιστή, εντοπίστε την ενότητα "Κλειδιά API" και κάντε κλικ σε αυτήν. Στη συνέχεια, κάντε κλικ στο κουμπί "Νέο κλειδί" για να δημιουργήσετε ένα νέο κλειδί API ειδικά για την εφαρμογή σας Android.

Βήμα 4: Δώστε τις απαραίτητες λεπτομέρειες
Δώστε το ακόλουθο όνομα για το κλειδί API σας: "GPTNOADS" για να σας βοηθήσει να προσδιορίσετε τον σκοπό του. Εάν θέλετε, μπορείτε να καθορίσετε τις σχετικές λεπτομέρειες χρήσης για την εφαρμογή σας GPTNOADS, όπως την περίπτωση χρήσης για την οποία προορίζεται και οποιαδήποτε άλλη σχετική πληροφορία.

Βήμα 5: Ελέγξτε και αποδεχτείτε τους όρους του OpenAI
Φροντίστε να διαβάσετε διεξοδικά τους όρους παροχής υπηρεσιών και την πολιτική αποδεκτής χρήσης του OpenAI. Αφού κατανοήσετε και συμφωνήσετε με τους όρους, προχωρήστε με την αναγνώριση της συμμόρφωσής σας.

Βήμα 6: Ανακτήστε το κλειδί API σας
Μετά την αποδοχή των όρων, θα δημιουργηθεί το κλειδί API σας. Φροντίστε να αντιγράψετε και να αποθηκεύσετε με ασφάλεια αυτό το κλειδί, καθώς παρέχει πρόσβαση στο ChatGPT API.

Βήμα 7: Ενσωματώστε το κλειδί API στην εφαρμογή GPTNOADS
Στο μενού GPTNOADS επιλέξτε "κλειδί API" και μετά πληκτρολογήστε μη αυτόματα το κλειδί.

Βήμα 8: Δοκιμάστε και βελτιώστε την ενσωμάτωσή σας
Με το κλειδί API ενσωματωμένο στην εφαρμογή GPTNOADS, ξεκινήστε τις δοκιμές με μια απλή ερώτηση όπως για παράδειγμα: «Η πρωτεύουσα της Γαλλίας» και παρατηρήστε τις απαντήσεις. Σε περίπτωση μηνύματος σφάλματος, ελέγξτε αν πληκτρολογήσατε το σωστό κλειδί. Είναι ένας μακρύς αλφαριθμητικός κωδικός, επομένως μπορεί να αποτύχει η σωστή εισαγωγή του, ελέγξτε τη διαφορά μεταξύ μηδέν (0) και του γράμματος Ο. Μόλις εισαχθεί το κλειδί, θα δείτε την απάντηση στην οθόνη σας.

παράμετροι
Η πρώτη επιλογή αναφέρεται στο μοντέλο, υπάρχουν δύο μοντέλα:
gpt-4 και gpt-3.5-turbo
Το gpt-4 είναι το πιο ισχυρό μοντέλο αλλά και χαμηλότερο και πιο ακριβό σε σύγκριση με το gpt-3.5-turbo
Το gpt-3.5-turbo είναι γρήγορο, λιγότερο ακριβό, αλλά είναι λιγότερο ακριβές από το gpt-4
Τα μοντέλα εξελίσσονται γρήγορα, επομένως είναι καλύτερο να ανατρέξετε στις επίσημες σελίδες
Πλήρης επεξήγηση των μοντέλων στην αγγλική γλώσσα στη διεύθυνση: https://platform.openai.com/docs/models/overview
Επεξηγήσεις σχετικά με το κόστος στα αγγλικά υπάρχουν στο ίδιο έγγραφο στην παράγραφο "Tokens".
Άλλες παράμετροι είναι διαθέσιμες για επαγγελματίες που θέλουν να τις δοκιμάσουν. Χρησιμοποιήστε με προσοχή γιατί είναι απαραίτητο να γνωρίζετε πώς λειτουργούν για την αποφυγή διαταραχών. Μην τα χρησιμοποιείτε εάν δεν είστε σίγουροι πώς να το κάνετε.
Πλήρης επεξήγηση των παραμέτρων στην αγγλική γλώσσα στη διεύθυνση:
https://platform.openai.com/docs/api-reference/completions/create

Κλειδί API
Εδώ μπορείτε να εισαγάγετε το προσωπικό σας κλειδί API. Ανατρέξτε στις επεξηγήσεις πάνω σε αυτήν την ενότητα βοήθειας για να μάθετε πώς να αποκτήσετε το κλειδί API.
Σελίδα όπου μπορείτε να εγγραφείτε στη διεύθυνση:
https://openai.com/
Αφού συνδεθείτε, αποκτήστε το κλειδί API στη διεύθυνση: https://platform.openai.com/
Κάντε κλικ στο όνομα του λογαριασμού σας που βρίσκεται συνήθως πάνω δεξιά στην οθόνη, επιλέξτε "Προβολή κλειδιών API" και δημιουργήστε το κλειδί API σας.
