Öncelikle aşağıdaki adımları izleyerek ChatGPT'yi GPTNOADS uygulamasında kullanmak için bir API anahtarı edinin:

1. Adım: Bir OpenAI hesabı için kaydolun
Henüz yapmadıysanız, OpenAI web sitesini (https://openai.com) ziyaret edin ve bir hesap oluşturun. Gerekli bilgileri sağlayın ve kayıt işlemini tamamlayın.

2. Adım: OpenAI Developer Dashboard'a gidin
OpenAI hesabınıza başarıyla kaydolup oturum açtıktan sonra, OpenAI Developer Dashboard'a gidin. Burası, API anahtarlarınızı yönetebileceğiniz ve ilgili bilgilere erişebileceğiniz yerdir.

3. Adım: Yeni bir API anahtarı oluşturun
Geliştirici Kontrol Panelinde "API Anahtarları" bölümünü bulun ve tıklayın. Ardından, özellikle Android uygulamanız için yeni bir API anahtarı oluşturmak üzere "Yeni Anahtar" düğmesini tıklayın.

4. Adım: Gerekli ayrıntıları sağlayın
API anahtarınız için şu adı sağlayın: Amacını belirlemenize yardımcı olması için "GPTNOADS". İsterseniz GPTNOADS uygulamanız için kullanım amacı ve diğer ilgili bilgiler gibi ilgili kullanım ayrıntılarını belirtebilirsiniz.

5. Adım: OpenAI şartlarını inceleyin ve kabul edin
OpenAI'nin hizmet şartlarını ve kabul edilebilir kullanım politikasını baştan sona incelediğinizden emin olun. Şartları anlayıp kabul ettikten sonra, uygunluğunuzu onaylayarak devam edin.

6. Adım: API anahtarınızı alın
Şartları kabul ettikten sonra, API anahtarınız oluşturulacaktır. ChatGPT API'ye erişim izni verdiği için bu anahtarı kopyalayıp güvenli bir şekilde sakladığınızdan emin olun.

7. Adım: API anahtarını GPTNOADS uygulamanıza entegre edin
GPTNOADS menüsünde "API anahtarı"nı seçin, ardından anahtarı manuel olarak girin.

8. Adım: Entegrasyonunuzu test edin ve hassaslaştırın
GPTNOADS uygulamasına entegre edilen API anahtarıyla, örneğin "Fransa'nın başkenti" gibi basit bir soruyla teste başlayın ve gözlemleyin cevap. Hata mesajı durumunda doğru anahtarı girip girmediğinizi kontrol edin. Bu uzun alfasayısal bir koddur, bu nedenle doğru girilmeyebilir, sıfır (0) ile O harfi arasındaki farkı kontrol edin. Anahtar girildikten sonra yanıtı ekranınızda göreceksiniz.

parametreler
İlk seçim modeli ifade eder, iki model vardır:
gpt-4 ve gpt-3.5-turbo
gpt-4 en güçlü modeldir ancak aynı zamanda gpt-3.5-turbo'ya kıyasla daha düşük ve daha pahalıdır
gpt-3.5-turbo hızlıdır, daha ucuzdur ancak gpt-4'ten daha az isabetlidir
Modeller hızla gelişiyor, bu nedenle resmi sayfalara başvurmak daha iyi
Modellerin İngilizce dilinde tam açıklaması: https://platform.openai.com/docs/models/overview
Maliyetlerle ilgili İngilizce açıklamalar aynı belgede "Tokenlar" paragrafında yer almaktadır.
Diğer parametreler, bunları test etmek isteyen profesyoneller için mevcuttur. Kesintileri önlemek için nasıl çalıştıklarını bilmek gerektiğinden dikkatli kullanın. Nasıl yapacağınızdan emin değilseniz bunları kullanmayın.
İngilizce dilinde parametrelerin tam açıklaması:
https://platform.openai.com/docs/api-reference/completions/create

Kayıt olabileceğiniz sayfa:
https://openai.com/
Oturum açtıktan sonra API anahtarını şu adresten edinin: https://platform.openai.com/
Genellikle ekranın sağ üst kısmında bulunan hesap adınıza tıklayın, "API anahtarlarını görüntüle"yi seçin ve API anahtarınızı oluşturun.
