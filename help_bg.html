Прежде всего, получите ключ API для использования ChatGPT в приложении GPTNOADS, выполнив следующие действия:

Стъпка 1: Регистрирайте се за OpenAI акаунт
Ако още не сте го направили, посетете уебсайта на OpenAI (https://openai.com) и създайте акаунт. Предоставете необходимата информация и завършете процеса на регистрация.

Стъпка 2: Отидете до таблото за управление на разработчици на OpenAI
След като успешно се регистрирате и влезете във вашия OpenAI акаунт, отворете таблото за управление на OpenAI Developer. Това е мястото, където можете да управлявате своите API ключове и да получите достъп до съответната информация.

Стъпка 3: Създайте нов API ключ
На таблото за разработчици намерете секцията „API ключове“ и щракнете върху нея. След това щракнете върху бутона „Нов ключ“, за да генерирате нов API ключ специално за вашето приложение за Android.

Стъпка 4: Предоставете необходимите подробности
Въведете следното име за вашия API ключ: „GPTNOADS“, за да ви помогне да идентифицирате неговата цел. Ако желаете, можете да посочите съответните подробности за употребата на вашето приложение GPTNOADS, като например предвидения случай на употреба и всякаква друга подходяща информация.

Стъпка 5: Прегледайте и приемете условията на OpenAI
Уверете се, че сте прегледали внимателно условията на услугата и правилата за приемлива употреба на OpenAI. След като сте разбрали и сте се съгласили с условията, продължете с потвърждаване на съответствието ви.

Стъпка 6: Извлечете вашия API ключ
След като приемете условията, вашият API ключ ще бъде генериран. Не забравяйте да копирате и съхранявате сигурно този ключ, тъй като той предоставя достъп до API на ChatGPT.

Стъпка 7: Интегрирайте API ключа във вашето приложение GPTNOADS
В менюто GPTNOADS изберете „API ключ“, след което въведете ръчно ключа.

Стъпка 8: Тествайте и прецизирайте вашата интеграция
С API ключа, интегриран в приложението GPTNOADS, започнете да тествате с прост въпрос като например: „Столицата на Франция“ и наблюдавайте отговорите. В случай на съобщение за грешка проверете дали сте въвели правилния ключ. Това е дълъг буквено-цифров код, поради което може да се случи да не го въведете правилно, проверете разликата между нула (0) и буквата O. След като ключът бъде въведен, ще видите отговора на екрана.

параметри
Първият избор се отнася до модела, има два модела:
gpt-4 и gpt-3.5-turbo
gpt-4 е най-мощният модел, но и по-нисък и по-скъп в сравнение с gpt-3.5-turbo
gpt-3.5-turbo е бърз, по-евтин, но е по-малко точен от gpt-4
Моделите се развиват бързо, затова е по-добре да се обърнете към официалните страници
Пълно обяснение на моделите на английски език на: https://platform.openai.com/docs/models/overview
Обясненията за разходите на английски са в същия документ в параграф „Жетони“.
Други параметри са достъпни за професионалисти, които желаят да ги тестват. Използвайте с повишено внимание, защото е необходимо да знаете как работят, за да избегнете смущения. Не ги използвайте, ако не сте сигурни как да го направите.
